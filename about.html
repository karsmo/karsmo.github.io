How to work with VisuNet?

Input format

VisuNet is available for any rule-based classifier in supported data frame formats.

‘Line by line’ format

Data should be expressed as data frame and needs to contain the following columns:

  • FEATURES - the left-hand side of the rule corresponds to comma-separated attributes and their values, factor
  • DECISION - the right hand side of the rule corresponds to decision value, factor
  • ACC_RHS - the rule accuracy, numeric
  • SUPP_RHS - the rule support, numeric

A p-value (‘PVAL’) calumn is not obligatory.

Sample ‘line by line’ data frame
FEATURES DECISION ACC_RHS SUPP_RHS PVAL
4 NPR2=2,CAPS2=3 control 1.00000 17 3.97e-05
5 MAP7=2,COX2=3 autism 1.00000 23 6.77e-05
6 NCKAP5L=1,PPOX=1 control 0.95119 20 6.89e-05

You can use ‘line by line’ format with the option: type = 'L

rules = data(autcon_ruleset)
vis_out = visunet(rules, type = 'L')

R.ROSETTA output

Rule data frame that is the output of R.ROSETTA can be directly run in VisuNet. See ?rosetta from the R.ROSETTA package for details.

The R.ROSETTA output format can be use with the option: type = 'RDF

resultsRos <- rosetta(autcon)
vis_out = visunet(resultsRos$main, type = 'RDF')
